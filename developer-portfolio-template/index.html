February - March 2024

Arduino Solar Kit

Satakunta University Of Applied Sciences, Finland

The final model I developed incorporates a dynamic solar panel system that tracks sunlight using an LDR sensor. A servo motor adjusts the panel's orientation accordingly, ensuring optimal exposure to sunlight. An LCD display provides real-time feedback on the sun's position, while a buzzer signals each adjustment. Additionally, humidity and temperature sensors provide continuous environmental monitoring. A button interface enables on-demand access to humidity levels, enhancing user interaction and control. This comprehensive design showcases my proficiency in integrating multiple sensors and actuators into a cohesive system, bridging hardware and software components seamlessly.

Autonomous Navigation Robot

Satakunta University Of Applied Sciences, Finland

In my current ROS project, I am developing a robot car equipped with four wheels capable of navigating through obstacles autonomously. The car utilizes ultrasonic sensors to detect obstacles in its path and a camera for visual perception. By integrating ROS (Robot Operating System), I am implementing algorithms for obstacle avoidance and path planning, enabling the robot to navigate efficiently and safely in dynamic environments. This project demonstrates my expertise in robotics, sensor integration, and software development using ROS, contributing to advancements in autonomous vehicle technology.

The Software Developer Is Dead: Long Live The Software Developer

In Anthony Hughes’ article for Forbes, "The Software Developer Is Dead: Long Live The Software Developer," the notion that software development is dying is thoroughly debunked. Despite concerns over AI like ChatGPT and recent tech layoffs, the demand for software developers remains robust. Hughes emphasizes that digital transformation continues to drive the need for developers across various industries, including finance, healthcare, and manufacturing. The field is evolving, with developers now using more advanced tools, but their expertise remains essential. As technology progresses, new opportunities in coding and tech careers will emerge, ensuring that software developers are indispensable for solving complex problems and driving innovation.
